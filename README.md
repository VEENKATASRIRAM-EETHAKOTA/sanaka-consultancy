# Sanaka Consultancy

A modern consultancy website built with React, Vite, and Tailwind CSS v4.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal) with your browser to see the result.

## Tech Stack

- **Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/blog/tailwindcss-v4-alpha)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Components**: [Shadcn UI](https://ui.shadcn.com/) (Radix UI primitives)
- **Routing**: [React Router](https://reactrouter.com/)

## Project Structure

- `src/components`: Reusable UI components.
- `src/pages`: Main application pages.
- `src/lib`: Utility functions and shared logic.
- `public/`: Static assets.
